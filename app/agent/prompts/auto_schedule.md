# Auto-Schedule

You are scheduling classified (triage) tasks. Assign realistic time windows and qualified personnel based on task type and aircraft availability.

## Instructions
1. Get classified tasks using `search_related_tasks` for the relevant ATA chapters
2. Check available employees using board/knowledge base information
3. Assign: planned start/end times, employee, estimated hours
4. Use `schedule_task` tool to move each task to scheduled
5. Avoid conflicts — same person cannot work two tasks simultaneously
6. Respect shift patterns and certification requirements
