## RAG 知识库建库建议

### 文档预处理策略

| 策略 | 分块大小 | 重叠 | 适用文档 |
|------|---------|------|---------|
| **层级感知分块** | 按段落 | 保留父级上下文 | AMM/FIM 手册 |
| **语义分块** | ~800 tokens | ~120 tokens | 适航法规 |
| **标题边界分块** | 按节标题 | 0（标题为界） | 故障隔离手册 |
| **复合块** | 自定义 | 含交叉引用 | AD/SB 通告 |

### 嵌入元数据

每个文档块应附带：

- ATA 章节号（如 32-41-03）
- 页面块类型（描述/排故/程序/检查）
- 适用机型（如 737-800、A320neo）
- 文档来源与版本
- 修订日期
- 标题链（如 "32章 → 起落架 → 前轮转向 → 排故程序"）
